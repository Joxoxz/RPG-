# RPG Hotseat VTT (Offline)

Plataforma web de RPG de mesa (Virtual Tabletop) 100% client-side.

## Como usar

1. Baixe/clone os arquivos.
2. Abra `index.html` em um navegador moderno.
3. Pronto: tudo funciona offline (sem servidor e sem internet).

## Recursos principais

- Multiplayer local hotseat com troca de turno.
- Upload de mapa PNG/JPG.
- Tokens com ficha, HP, avatar PNG em base64 e drag & drop.
- Grid configurável com snap opcional.
- Medidor de distância.
- Fog of War com revelar/ocultar.
- Rolador de dados com histórico.
- Salvamento local via LocalStorage (campanha completa).
